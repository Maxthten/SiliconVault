# SiliconVault v1.4.0 Beta

本次版本重点提升了数据安全、稳定性和使用体验。

## 主要更新

- 开发版与正式版数据库完全隔离，避免测试数据影响正式数据。
- 完善操作日志、撤销机制、消耗统计和批量库存记录。
- 加强备份导入、资源文件、临时目录及中文路径安全。
- 修复 BOM 强制扣除失败，允许确认后产生负库存，普通出库仍受库存限制。
- 优化新增/编辑元件、BOM 项目和分类布局配置界面。
- 恢复自然的整卡拖拽，并修复字段覆盖、按钮误触等交互问题。
- 优化搜索、库存统计、主题切换、路由和图表加载性能。
- 完善高级维护功能、类型定义及数据库和导入备份测试。

## 安装包

- Windows x64：`.exe`
- macOS Intel：`x64.dmg`
- macOS Apple Silicon：`arm64.dmg`

> 当前版本为 Beta。Windows 和 macOS 安装包暂未进行商业代码签名，系统可能显示安全提示。
